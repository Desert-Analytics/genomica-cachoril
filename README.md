El objetivo principal de esta investigación consiste en **describir la relación taxonómica molecular de poblaciones de la lagartija cola de cebra (Callisaurus draconoides) en localidades de Sonora, México.** En este trabajo se incluye el desarrollo de código estructurado, versionado y reproducible para la investigación del cliente; entre los desarrollos se incluye un análisis morfométrico de las especies de **C. draconoides** mediante Análisis de Varianza (ANOVA), asi como también Análisis Molecular de Varianza (AMOVA). Adicionalmente se realizaron estudios genéticos de la especie mediante el desarrollo de Árboles Filogenéticos, Análisis de Componentes Principales (PCA) y un estudio de la distribución de las secuencias genéticas mediante Bootstrap.

Cabe destacar que en el presente trabajo se utilizaron Python y R como lenguajes de programación y desarrollo de código.
